Add a map key

The basemap is drawn with your own public map key. Pick MapTiler or Mapbox; the key stays in this browser only (localStorage) and is sent only to the provider you choose.

  1. MapTiler keys come from Cloud keys — the quickest to set up.
  2. Mapbox keys come from account tokens and start with pk..
  3. Paste the public key below.
